功能说明：

显示每行从开头算起 num1 到 num2 的文字。

语　　法：

cut -cnum1-num2 filename

补充说明：

显示每行从开头算起 num1 到 num2 的文字。

参　　数：

范例：

        shell>> cat example
        test2
        this is test1
        shell >> cut -c0-6 example       ## print 开头算起前 6 个字元
        test2
        this i

示例：
设ml和m2是当前目录下的两个文件 cat m1 （在屏幕上显示文件ml的内容） cat m1 m2 （同时显示文件ml和m2的内容） cat m1 m2 > file （将文件ml和m2合并后放入文件file中）
